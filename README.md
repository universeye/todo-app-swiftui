# Features

* Fully programmatically UI (SwiftUI)
* Random Empty View
* Animated add button
* Be able to choose alternate icons
* First responder textfield with UIRepresentable
* Ability to change app's overall theme
* iPad/Mac support
* Dark/Light mode support
